# GDScript教程

`GDScript`是`Godot`引擎的专用脚本语言，具有简单易学、开发便捷的特点。本教程将带你从基础代码框架入门，逐步掌握`GDScript`语法，学习面向对象编程和设计模式，并提供了丰富的代码示例。

文档已部署为网站，可以[点击阅读](https://godothub.com/oss/gdscript-tutorial)。


## 主要内容

1. GDScript语法
2. 可调用体和信号
3. 面向对象入门
4. 设计模式
5. 注解和文档注释
